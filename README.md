# CANDIDATES

Личный кабинет кандидата с матрицей навыков и списком интервью

## Возможности

### Страница входа и регистрации

* Чтобы зайти на страницу, пользователь должен быть зарегистрирован
* Если он не регистрировался, то ему предлагается зарегистрироваться
* Идет проверка валидного заполнения полей (пустые значения, мин длина пароля, вид почты, повтор пароля)
* Если пользователь зарегистрировался, то он переходит на страницу Dashboard
* При этом в localStorage храним почту пользователя и статус регистрации

### Страница Dashboard

* Левая панель - информация о пользователе через localStorage (его почта)
* Проверка авторизации через authState
* Осуществлена имитация задержки - спиннер загрузки и скелетон
* Выбор роли через выпадающий список и сохранение ее в localStorage через кнопку
* Кликабельная таблица интервью - можно посмотреть подробности
* Слева навигационный блок, чтобы перемещаться между страницами, и кнопка выхода

### Страница Детали интервью

* Левая панель статична, на ней расположена кнопка возврата на Dashboard
* Осуществлена имитация задержки - загрузка спиннер
* Получение ID интервью из URL параметров
* Сюда можно попасть только при нажатии на какую-нибудь строку таблицы в Dashboard
* Редирект при ненайденном интервью
* Рассматриваются случаи когда есть данные об интервью - блоки доп информации
* Рассматриваются случаи, когда интервью еще запланировано - блоки доп информации пусты

### Страница Матрица навыков

* Левая панель - информация о пользователе (его почта)
* Осуществлена имитация задержки - спиннер загрузки и скелетон
* Слева навигационный блок, чтобы перемещаться между страницами, и кнопка выхода
* Есть фильтр категорий навыков
* Разница целевого и текущего уровня навыков имеет цветовую индикацию для лучшей визуализации
* Проверка авторизации через localStorage
* Обработка ошибок загрузки с fallback-данными
